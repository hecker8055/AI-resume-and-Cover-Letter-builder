

# AI-resume-and-Cover-Letter-builder
I’ve developed an innovative AI-powered platform that tackles the most pressing challenges faced by today’s job seekers. By integrating the Gemini AI API, along with advanced Natural Language Processing (NLP) and Machine Learning (ML) technologies, our platform transforms the often time-consuming and frustrating process of creating job applications into a streamlined, intelligent, and empowering experience.

Whether it's crafting tailored resumes, writing personalized cover letters, or optimizing responses to job descriptions, our solution makes the entire process faster, smarter, and more effective—so users can focus on what matters: landing their next opportunity.

## Run Locally

Clone the project

```bash
  git clone https://github.com/hecker8055/AI-resume-and-Cover-Letter-builder.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
